# TheAltText Browser Extension

Chrome/Edge extension for real-time alt text suggestions on any webpage.

## Features

- **Right-click any image** to generate WCAG-compliant alt text
- **Scan entire pages** to highlight images missing alt text
- **Copy generated alt text** to clipboard with one click
- Warm earthy theme matching the main TheAltText app

## Installation (Developer Mode)

1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" (top right)
3. Click "Load unpacked"
4. Select this `extension/` directory

## Setup

1. Click the TheAltText extension icon
2. Enter your TheAltText API URL
3. Enter your API key (get one from the Developer page in the app)
4. Click "Save Settings"

## Usage

- **Right-click** any image and select "Generate Alt Text with TheAltText"
- Click the extension icon and press "Scan This Page" to audit the current page
